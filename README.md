# Utilization Calculator

A simple Angular-based tool to calculate the payment required to achieve 
a specific credit card utilization percentage, helping users manage their 
credit score effectively.

🔗 **Live Demo:** [utilizationcalculator.netlify.app](https://utilizationcalculator.netlify.app/)

## Features
- Calculate the exact payment needed to hit a target utilization percentage
- Helps users strategically manage their credit score
- Clean, simple UI for quick calculations

## Tech Stack
- Angular 16
- TypeScript
- HTML/CSS

## Getting Started

### Prerequisites
- Node.js
- Angular CLI (`npm install -g @angular/cli`)

### Installation
1. Clone the repo:
   `git clone https://github.com/lillianmg/UtilizationCalculator`
2. Install dependencies:
   `npm install`
3. Run the dev server:
   `ng serve`
4. Navigate to `http://localhost:4200`
